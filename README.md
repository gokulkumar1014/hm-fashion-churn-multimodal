# HM Fashion Churn Multimodal

This repository contains the backend and frontend for the HM Fashion Churn Multimodal project.

## Directory Structure
```text
hm-fashion-churn-multimodal/
├── backend/            # FastAPI + ONNX Brain
│   ├── app/            # Source code
│   ├── assets/         # THE 11 SOUL FILES (from your zip)
│   ├── Dockerfile      # For Cloud Run
│   └── requirements.txt
├── frontend/           # React + Vite UI
│   ├── src/            # Components & Logic
│   └── package.json
├── docker-compose.yml  # To run everything locally with one command
└── README.md
```
